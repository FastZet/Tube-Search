Configure Tube Search Add-on

This add-on requires a YouTube Data API Key to function. Please obtain your key and enter it below to generate your personalized installation link.

How to get your YouTube Data API Key:

  1. Go to the Google Cloud Console API Dashboard.
  2. Log in with your Google account.
  3. Create a new project (or select an existing one).
  4. In the "APIs & Services" section, go to "Library".
  5. Search for "YouTube Data API v3" and enable it for your project.
  6. Go to "Credentials" on the left sidebar.
  7. Click "CREATE CREDENTIALS" and choose "API Key".
  8. Copy the generated API key.
  9. **Important:** Restrict your API key to "HTTP referrers (web sites)" if you intend to secure it, but for Stremio, it's often used client-side. For this add-on, it's passed server-side. For simplicity, just generating the key is enough for initial setup.
