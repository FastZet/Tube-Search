YouTube Stream Provider Configuration

This add-on provides YouTube streams for movies and shows. It requires API keys and allows you to customize stream results.

How to get your API Keys:

YouTube Data API Key:

  1. Go to the Google Cloud Console API Dashboard.
  2. Log in, create/select project.
  3. In "APIs & Services" -> "Library", search for "YouTube Data API v3" and enable it.
  4. Go to "Credentials", click "CREATE CREDENTIALS" and choose "API Key". Copy it.

TMDb API Key (v3):

  1. Go to The Movie Database (TMDb) and create an account.
  2. Log in, then go to your API Settings page.
  3. Request a new API Key (v3 Developer).
  4. Agree to terms and fill in the application details. Copy your generated API Key.

Important: Restrict your API keys in their respective consoles for security.
