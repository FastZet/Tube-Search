Tube Search Add-on

This Stremio add-on allows you to search YouTube videos with advanced filters for duration and resolution directly within Stremio.

Important Information:

This add-on requires a **YouTube Data API Key** to function. This key is provided by the add-on's deployer (administrator), not by individual users.

If you are the deployer, ensure the `YOUTUBE_API_KEY` environment variable is set in your deployment environment (e.g., Hugging Face Space secrets).

For more details or to report issues, please visit the add-on's GitHub repository.
